# Chatbot with PDF Integration using AWS Bedrock, LangChain, and Streamlit  

This project demonstrates a chatbot application that processes PDF documents to provide intelligent and contextual responses. It leverages AWS Bedrock, LangChain, FAISS for vector storage, and various Large Language Models (LLMs) like Claude, Titan, and LLaMA. The interface is built using Streamlit, enabling an intuitive user experience.  

---

## Features  
- **PDF Document Parsing**: Upload a PDF to extract and process its content.  
- **Vector Storage**: Utilize FAISS for efficient similarity search on text embeddings.  
- **Embedding Models**: Use Amazon Titan for generating embeddings.  
- **LLMs**: Chatbot powered by models like Claude, LLaMA, and others via AWS Bedrock.  
- **Streamlit Interface**: A user-friendly interface for seamless interaction.  

---

## Technologies Used  
- **Programming Language**: Python  
- **Frameworks & Libraries**:  
  - [LangChain](https://langchain.com)  
  - [FAISS](https://github.com/facebookresearch/faiss)  
  - [Streamlit](https://streamlit.io)  
- **Cloud Service**: AWS Bedrock  
- **LLMs**: Claude, Titan, LLaMA
